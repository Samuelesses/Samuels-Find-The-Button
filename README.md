# Greg And The Button
Greg and the Button is a story game where you play as the character ‘Greg’. Similar to ‘The Stanley Parable,’ but in an indie ‘Find the button’ format. In this short game, you find buttons while metaphorically fighting with the narrator.

## Insperation
The Stanly Parable was a big insperation for this project, it guided the idea that the narrorator of a story can play a larger part than just narrorationg.
